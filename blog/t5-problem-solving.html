Explain to your non-tech friend

.map() - This method is similar to the .length method. It basically goes through all the elements in an array then creates a new array with modified values.

.filter() - This method goes through an array and returns any value that meets a certain condition.

.reduce() - This method takes two arrays and joins them into a single array by summing up all the value in the array and returns a new single array.

Recursion VS Iteration

Both processes are very similar in that both in that both solve problems by repeating a certain task. Iteration repeats a process x amount of times until it reaches a certain target. Recursion is the process of breaking the problem up into smaller pieces until you can solve it.
