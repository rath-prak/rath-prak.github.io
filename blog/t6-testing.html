What 'scope' is, and how it works in javascript?

parentheses ()

  1. Execute a function, e.g var result = doSomething();
  2. Assign function parameters, e.g function multiply(x, y){};
  3. Conditional statements, e.g for(var i = 0; i > 10; i++);
  4. Mathematics, e.g var x = 3 * (num - 1);

brackets []

  1. Declaring arrays, e.g var score = [2, 4, 6, 2];
  2. Accessing an array index e.g console.log(score[2]);
  3. Accessing an object using bracket notation e.g console.log(object[“item1”]);

braces {}

  1. Creating objects, e.g car = {};
  2. main body of a function, e.g function multiply(x, y){ ...do something… };

single quotes ‘’

  1. declaring a string variables, e.g var bird = “eagle”;

double quotes ""

  1. double quotes are used in the same way as single quotes
